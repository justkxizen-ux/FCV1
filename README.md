# FCV1
A unique approach to flight controller development for drones ( Quad Planes, quadcopters, tricopters,bicopters) , Cansats , Rocket flight controllers. This repo contains the pcb designs , software and all the details necessary to learn and explore the inner workings of a flight controller by giving flexibility to use any sensor of your choice (P.S Will be uploading the full firmware design once I finish up with it along with a detailed user guide).
